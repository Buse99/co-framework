This folder contains the following:

a5-diff-frame.S: Write your x86 diff program here.
test 1 - 7: Test folders that test the diff function.

Each test folder contains:
- Two files to test diff on.
- A file with input for running and expected output.

There are 7 tests. Each test is meant to test a specific part of the student's code.
These are as follows:

Test 1 - Basic alphabetical test
Test 2 - Longer alphabetical test
Test 3 - Numerical test
Test 4 - Test if diff recognizes upper case vs lower case
Test 5 - Test -i flag
Test 6 - Test -B flag
Test 7 - Test -i flag and test -B flag
